# React + TypeScript + Vite

## Version Node
v20.10.0

## Init
```bash
nvm use 20.10.0
yarn install
```

## Dependencies
Change environment url in .env.development file
```bash
yarn dev
```

## Production
Before compiling, change the environment url in the .env.production file.
```bash
yarn build
```

# Testing
```bash
yarn test
